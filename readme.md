# Smart URL Shortener

A professional, high-performance URL shortener application engineered with **FastAPI**, **SQLAlchemy**, and **Tailwind CSS**.

## Features
- **Smart Tracking:** Automatically logs visitor IP addresses and User-Agent data for every click.
- **Modern UI:** Responsive design using Tailwind CSS with professional glassmorphism and animated gradients.
- **Efficient:** Lightweight backend built for speed.

## Technologies
- **Backend:** Python, FastAPI
- **Database:** SQLite, SQLAlchemy
- **Frontend:** Tailwind CSS, Jinja2

## Setup
1. Clone the repository: `git clone <your-repo-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Start the server: `uvicorn app.main:app --reload`
4. Access the app: `http://127.0.0.1:8000`
